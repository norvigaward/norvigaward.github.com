What are the restrictions?

Since the SARA Hadoop service is multitenant, you have to keep others in mind and it's our job to make sure everybody gets a turn. Our policy is as follows:

  • You test your code locally
  • Once you're done you test your code on the testset on the cluster
  • You get no more than 2 runs on the complete dataset

We monitor all usage on the cluster, and if we notice somebody taking an unfair share of the cluster we will disqualify this person.

Where can I find the datasets?

Next to the Common Crawl set we provide two test sets. The three sets can be found at:

  • Testset on the VM: /home/participant/data/* (~163MB)
  • Testset on the cluster: /data/public/common-crawl/award/testset (~6.6GB)
  • Full set: /data/public/common-crawl/parse-output/* (~25TB)

What Pig example is included in the Virtual Machine?

The examples in the Virtual Machine come with an example pig script as well, to illustrate the use of the Pig Loader. This example does a simple count of HTTP status codes. It looks like this:

register /home/participant/git/commoncrawl-examples/lib/*.jar; 
register /home/participant/git/commoncrawl-examples/dist/lib/commoncrawl-examples-1.0.1.jar;
a = LOAD '/data/1346864466526_10.arc.gz' USING org.commoncrawl.pig.ArcLoader() as (date, length, type, statuscode, ipaddress, url, html, payload);
words = foreach a generate flatten(statuscode) as codes;
grpd = group words by codes; 
cntd = foreach grpd generate group, COUNT(words); 
dump cntd;

Why does it take so long before my job starts running at SARA?

The SARA Hadoop cluster is a multitenant cluster. This means you are sharing the cluster's resources with other users. If at other jobs are occupying all processing power of the cluster, your job will have to wait in queue until there is space again.

Where can I find information about jobs on the SARA cluster?

Both the namenode and jobtracker have a web interface. The Firefox browser in the VM already contains bookmarks to these pages.

You need to authenticate with kinit to get access to the web interfaces.
