Using MapReduce

Step 1: Starting Eclipse

When your Virtual Machine has started up, look on the left of your screen for the icon you see here. This is Eclipse, you can open it by clicking on it.

Step 2: Opening the org.commoncrawl.examples.ExampleTextWordCount example

When Eclipse has started, notice the Project Explorer on the left hand side of Eclipse. In here you see an entry with the title Common Crawl Examples. Expand the project by clicking on the arrow next to it, and do the same for the entry java, and then org.commoncrawl.examples. You should now see a list that includes:

  • ExampleArcMicroformat.java
  • ExampleMetadataDomainPageCount.java
  • ExampleMetadataStats.java
  • ExampleTextWordCount.java
  • SampleFilter.java

Double click on ExampleTextWordCount.java to open.

Step 3: Run the example

In the menu bar in the top of Eclipse you see an icon with a triangle, looking like a "play" button, with a little downward arrow next to it. Click the downward arrow and select "1. ExampleTextWordCount - local". The wordcount job now starts running, and you should see the output appearing on your screen.

Step 4: Explore

This job is described in the class ExampleTextWordCount. Notice the main() method at the bottom of the file. Then find the run() method above that, and start exploring it. See this entry in our FAQ for a list of resources that might help you in figuring out what this example does.

Step 5: Look at the other examples

Now also have a look at the other examples, and see if you can get them to run. Hint: also look at "Run → Run Configurations" in Eclipse.

Using Pig

Step 1: Start a terminal

When your Virtual Machine has started up, look on the left of your screen for the icon you see here. This is the terminal, you can open it by clicking on it.

Step 2: Compiling the code

Go to the directory that holds the examples code:

participant@hadoop-vm:~$ cd git/commoncrawl-examples

Use ant to compile the code

participant@hadoop-vm:~/git/commoncrawl-examples$ ant

Step 3: Run the example

Start pig in local mode and run the example.pig file:

participant@hadoop-vm:~/git/commoncrawl-examples$ pig -x local example.pig

Step 4: Explore

Open example.pig for reading, and see if you understand what's happening. See this entry in our FAQ for a list of resources that might help you in figuring out what this happens.
